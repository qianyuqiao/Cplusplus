### 构造函数和默认构造函数的区别？？
```
class A
{
public:
    A(){}; // 默认构造函数
    A(int a = 0){}; // 默认构造函数
    A(int a){}; // 构造函数
}
```
**一个类中可以有多个构造函数，但是只能有一个默认构造函数和一个析构函数**

```
{
```

```
    char* s;

public:
    Base(){};
```

### 基类的构造函数和析构函数，重载的赋值运算符，基类的友元不能被继承 
```
构造函数，析构函数，赋值运算符，友元
```

```

{
public:
int operator+(const A& b) { return vala + b.vala; } // 可以被继承
}
```

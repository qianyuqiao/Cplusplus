### 模板重载
```
template <typename T>
int add(T& a)
{
	return a + 1;
}

template <typename T>
int add(T* a, int n)
{
	return *a + 1 + n;
}
```

### 显式具体化
```
template <typename T>
int add(T& a)
{
	cout << "a+1" << endl;
	return a + 1;
}

template<> int add<int>(int&a)
{
	cout << "a+2" << endl;
	return a + 2;
}
```
结果:
```
a+2
```
